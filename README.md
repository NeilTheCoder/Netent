## Installation
### Windows
**Installing Netent bot on Widnows  
Requirements:  
`Windows 7 or Above  
NodeJS 8+  
Visual Studio`**  
**- How to install**  
**1. Install the file for Netent  
2. Change the file name `settings_example.json` to `settings.json` and change the Credentials inside it.  
3. Run the code through Visual Studio**
